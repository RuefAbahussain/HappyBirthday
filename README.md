# 🎈 Birthday Card

Create personalized birthday cards with photo upload, animations, balloons, and festive music.

=====================================

## Features

- Personalized Greeting - Enter sender name and recipient name, upload a photo of the birthday person
- Animated Party - Balloons, hearts, petals, confetti, and a party hat above the photo
- Birthday Music - Background music plays when you open the card

=====================================

## Deploy

1. Create new repository on GitHub
2. Upload all files
3. Go to Settings → Pages → Select "main" branch
4. Your card will be live at `https://yourusername.github.io/birthday-card`

=====================================

## Customize

### 🎵 Change Music
Replace the file in `music.mp3/` folder, then update line 188 in `script.js`:
```js
const src = 'music.mp3/your-song.mp3';
```
